Introduction

SkyHost is a fast, scalable cloud hosting platform for developers. Designed for simplicity and power, SkyHost allows you to deploy, manage, and scale your applications with a single command.

New to SkyHost? Start with our CLI tool for instant deployment and resource management.

What Is SkyHost?

SkyHost is a full-service platform that supports frontend, backend, containerized, and microservices-based applications. With GitHub integration, auto-scaling, and CLI-based control, you can deploy and manage apps in seconds.

Key Features

  • Seamless GitHub integration

System Requirements

  • Operating Systems: Windows 10+, macOS, Linux
  • Node.js v12 or higher
  • Internet access
  • Git (for deployment from repositories)

Installing the SkyHost CLI

The SkyHost CLI is the easiest way to interact with your cloud projects. Below are the installation methods for various package managers:

Using Homebrew (macOS):

$ brew install SkyHost-cli

Using NPM:

$ npm install -g SkyHost-cli

Using Yarn:

$ yarn global add SkyHost-cli

Next Steps

After installing the CLI, you're ready to initialize your project and deploy it to the cloud.

$ SkyHost init
$ SkyHost deploy

For advanced use cases, such as multi-region deployment, environment variables, or database integrations, refer to the Deployment section.
